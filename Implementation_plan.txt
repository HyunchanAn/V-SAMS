V-SAMS: 시각 기반 표면 분석 및 매칭 시스템 - 구현 계획
목표 설명 (Goal Description)
중간 가공품의 사진을 분석하여 모재 종류(Substrate Type)와 표면 마감(Surface Finish)을 식별하고, 이를 바탕으로 최적의 보호 필름을 추천하는 지능형 시스템을 개발합니다. 반사(Specular Reflection) 및 스케일 모호성 같은 난제를 해결하기 위해 멀티 헤드 분류 모델과 룰 기반 매핑 로직을 사용합니다.

## 부트스트래핑 전략 (핵심)
산업용 자체 데이터가 부족한 초기 단계이므로 오픈 데이터 부트스트래핑(Open Data Bootstrapping) 전략을 사용합니다:

## 재질 분류: MINC (Materials in Context) 데이터셋을 사용하여 금속, 플라스틱, 유리 등의 대분류를 사전 학습합니다.
## 텍스처 분석: DTD (Describable Textures Dataset) 데이터셋을 사용하여 표면 마감(반짝임, 거침, 무광 등)을 사전 학습합니다.
## 가상 DB: 초기 PoC(개념 증명) 단계에서는 복잡한 SQL 대신 JSON 기반의 가상 데이터베이스를 사용하여 추천 로직을 시연합니다.

아키텍처 (Architecture)

1. 이중 경로 시각 분석 (Dual-Path Visual Analysis)
    - 백본 (Backbone): ResNet50 (ImageNet 사전 학습 완료).
    - 헤드 1 (재질): MINC로 미세 조정 (클래스: Metal, Plastic, Glass 등).
    - 헤드 2 (표면 마감): DTD로 미세 조정 (클래스: Shiny, Rough, Porous 등).
    - 결합 (Fusion): 두 헤드의 결과를 결합하여 쿼리 프로필 생성 (예: Metal + Rough).

2. 비즈니스 로직 (Recommendation Engine)
    - 입력: 예측된 재질(Material), 예측된 마감(Finish).
    - 로직:
        시각적 특성을 물리적 특성으로 매핑 (예: Metal = 높은 표면 에너지).
        패턴을 제약 조건으로 매핑 (예: Rough = 높은 점착력 필요).
    - 데이터베이스: 제품 사양과 타겟 조건이 담긴 database.json.

3. 기술 스택 (Tech Stack)
    - AI/모델: PyTorch, timm, Albumentations.
    - 앱/데모 UI: Streamlit (Python 기반 웹 UI).
    - 백엔드: FastAPI (향후 API 분리 대비).
    - 인프라: Docker, WandB (실험 추적).
